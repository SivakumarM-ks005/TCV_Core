Steps to Send Email via Nodemailer and Gmail
Prepare Gmail Account:
Enable 2-Step Verification in your Google account security settings.
Generate an App Password. Go to security settings, search for "App Passwords," name it (e.g., "Nodemailer"), and copy the 16-character code.
Setup Node.js Project:
Initialize the project: npm init -y.
Install nodemailer: npm install nodemailer.
Write the Script:
Create a transporter using your Gmail address and the App Password. 

var transporter = nodemailer.createTransport({
    service: 'gmail',
    auth:{
        user: process.env.EMAIL,
        pass: process.env.PASSWORD
    }
});
